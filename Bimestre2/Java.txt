Una variable debe de tener 3 cosas fundamentales.

1. Identicador = nombre que le va a dar

2. Tipo de dato

3. Valor = Lo puedo obtener de dos formas.
	3.1 Asignandolo (con el simbolo de asignación que es el =)
	3.2 Leyendolo: pidiendolo al usuario (por medio
	de un objeto o una clase).
